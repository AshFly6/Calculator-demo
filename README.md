# 计算器 - Demo

可以直接无视本项目，本项目只是作者学习安卓应用开发的阶段性练习。本项目是支持 Android 4+ 的一款简单的的计算器软件，主界面参考了 Google 计算器。

## 已实现的功能

- 标准计算器
- 数学函数
- 没了……

但会持续更新——我会随着学习开发安卓应用逐步完善这个计算器。
